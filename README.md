# Fox Box
A 2D Rigidbody Physics Engine

## About

> "One learns from books and example only that certain things can be done. Actual learning requires that you do those things. – Frank Herbert, Children of Dune"

After developing several 2D physics engines throughout my university studies and personal projects, I decided to bring everything I have learned together into one complete package.

This project is intended to serve both as an educational resource and as an opportunity for me to refresh and expand my own understanding of 2D physics simulation.

## Project Goals

The engine is being designed with the following goals in mind:

* **Modularity** — Certain components can be used independently without requiring the entire engine.
* **Readability** — The code should be easy to understand, navigate, and learn from.
* **Minimal dependencies** — External dependencies will be reduced wherever practical. 
* **Extensibility** — Users should be able to replace or add their own algorithms within individual modules.
* **Flexibility** — You are free to use the full engine or take only the parts that are useful for your own projects.
